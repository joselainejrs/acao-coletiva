Mini projeto de ação coletiva

## Installation

#### Web
```
$ npm create react-app web 
```

###### Cliente http
```
$ npm install axios
```

######  Run the application
```
$ npm start
```

### Server
```
$ cd backend 
$ npm init -y
$ npm init -y
$ npm install express nodemon cors

```
######  Sql

```
$ npm install knex
$ npm install sqlite3
```
######  Run the application 
```
$ npm start 
```
